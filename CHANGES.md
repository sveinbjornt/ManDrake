# ManDrake Version History

## 10/11/2025 - Version 3.3.1

* Universal Intel/ARM 64-bit build
* No longer relies on groff, which is no longer included in macOS
* Now uses the mandoc binary shipping with macOS
* PDF export removed, since it relied on groff
* App is now notarized by Apple
* New application icon

## 30/03/2019 - Version 3.3

* Fixed issue with rendering preview for very large mdoc files
* ManDrake is now a Developer ID signed application
* Minor fixes to improve compatibility with Mojave's Dark Mode

## 15/10/2018 - Version 3.2

* Added support for macOS 10.14 Mojave Dark Mode

## 18/12/2016 - Version 3.1

* Fixed crash issue on macOS 10.12 "Sierra"

## 26/01/2016 - Version 3.0

* Overhauled user interface
* New, powerful, customizable ace text editor
* Automatic mdoc syntax checking with inline warnings in editor
* Better syntax highlighting
* Faster preview generation
* Export to HTML and PDF
* New application and document icons
* Updated documentation
* Sparkle autoupdating
* Code is now BSD licensed
* Better man templates

## 16/09/2011 - Version 2.1

* Updated user interface

## 28/07/2010 - Version 1.2

* Web preview field now retains scroll position.

## 30/07/2008 - Version 1.1

* Optimized live preview by using cat2html and WebKit instead of PDF rendering

## 11/06/2004 - Version 1.0

* Initial release
